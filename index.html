在线演示

  • demo.btlike.com

特性

  • 高性能: 5美金的单核,768MB的VPS上，每秒处理UDP请求超过12K，内存占用不超过100MB，每天抓取数千万去重infohash
  • 大容量: 采用Mysql分表存储,分16张表,设计容量6千万~8千万数据
  • 支持BEP0009
  • 全文搜索：采用elasticsearch实现全文索引和热度排序

组件

  • spider 底层DHT网络爬虫库
  • crawl 从DHT网络中抓取活跃的infohash并存储
  • storage 根据infohash从网络中抓取torrent并存储metadata
  • api 对外提供接口，响应搜索等请求
  • site web网站

安装准备

  • 安装golang
  • 安装mysql
  • 安装elasticsearch

安装组件

  • crawl
    go get github.com/btlike/crawl
  • api
    go get github.com/btlike/api
  • storage
    go get github.com/btlike/storage
  • site
    git clone https://github.com/btlike/site.git

设计思路

  • 从全网尽可能多的获取infohash
  • 可配置从DHT网络，通过BEP0009获取资源。大部分通过资源库查询，采用流量+带宽暴力遍历
  • 极力优化内存、CPU与带宽资源
  • 随着热门资源大部分被收录，每日更新资源数从数十万逐步降低到十来万
