btlike BT搜索引擎

项目文档

组件

  • crawl 从DHT网络中抓取活跃的infohash
  • storage 根据infohash从网络中抓取torrent并存储metadata
  • api 对外提供接口，响应搜索等请求
  • site web网站

安装准备

  • 安装golang
  • 安装mysql
  • 安装elasticsearch

安装组件

  • crawl: go get github.com/btlike/crawl
  • api: go get github.com/btlike/api
  • storage: go get github.com/btlike/storage
  • site: git clone https://github.com/btlike/site.git

注意

流量消耗较大
